drops
=====

A weee little jquery plugin to easily add clickable drop menu's to your website.

Basic Usage
```
<div class="btn">
    <a href="#">Click Me :) </a>
    <div class="dropdown">
        <ul>
            <li>Link</li>
            <li>Link</li>
            <li>Link</li>
            <li>Link</li>
        </ul>            
    </div>
</div>

Enjoy :)
